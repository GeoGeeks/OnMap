# OnMap
Esta aplicación fue desarrollada por el grupo Semillero de Innovación Geográfica (SIG) de Esri Colombia. Para mayor información consulte www.esri.co o visite @geo_geeks en twitter.
OnMap, es una aplicación que le permite visualizar y editar sus mapas web de ArcGIS. Agregue Url's de cuetas de ArcGIS o use el portal de ArcGIS Online para conectarse.
Una vez haya validado sus credenciales de ArcGIS, usted podrá referenciar su ubicación, realizar trazado de rutas encontrar direcciones y agregar nuevos puntos a sus mapas de ArcGIS.

## Instrucciones 

Verifique que la configuración del sistema sea la correcta para el API de Windows Phone en el ArcGIS Runtime SDK for .NET 10.2.5:
  * [Windows Phone](http://developers.arcgis.com/net/store/guide/system-requirements.htm).
  * [ArcGIS Runtime SDK for .NET](http://esriurl.com/dotnetsdk)
